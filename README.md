# Taskmate 📝

Taskmate is a sleek, lightweight task management application for Android designed to help users organize their daily routines efficiently. The app follows modern Android development standards, providing a seamless user experience with an immersive interface.

## ✨ Features

- **Task Persistence:** Uses **Room Database** to save your tasks locally so you never lose your data.
- **Material Design 3:** Implements the latest Google Material You components for a modern look and feel.
- **Edge-to-Edge Experience:** Fully immersive UI where content flows behind the system status and navigation bars.
- **Splash Screen:** Uses the official `androidx.core:core-splashscreen` for a smooth app launch.
- **Responsive Layouts:** Built using `ConstraintLayout` to ensure compatibility across different screen sizes.
- **Developer Info:** Dedicated section providing app versioning and developer details.

## 🛠 Tech Stack & Libraries

- **Language:** Java
- **UI Framework:** Material Components (M3)
- **Architecture:** MVVM (Model-View-ViewModel) with LiveData
- **View Binding:** For type-safe interaction with UI elements.
- **Local Database:** Room Persistence Library
- **System UI:** AndroidX Activity Edge-to-Edge & WindowInsets
